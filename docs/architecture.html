Request / Response Data Flow

  1. User changes a filter or navigates to a view in the Vue SPA.
  2. Component reads shared state via useFilters() composable.
  3. api.js builds query params and dispatches an Axios GET.
  4. Request hits a FastAPI route in server/main.py (e.g. /api/orders).
  5. Handler invokes mock_data.py to access in-memory arrays.
  6. apply_filters() / filter_by_month() narrow the result set.
  7. Pydantic response_model validates and serializes the payload.
  8. Axios resolves; component updates reactive refs.
  9. Computed properties derive UI state; Vue re-renders the template.
